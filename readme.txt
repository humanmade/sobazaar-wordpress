Show off your fashion boards from SOBAZAAR!

== Description ==

SOBAZAAR is a social network for fashion lovers, where you can create your own fashion boards.
This plugin lets you display your SOBAZAAR fashion boards with the use of a simple WordPress shortcode.

== Installation ==

1. Upload the `sobazaar` plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
